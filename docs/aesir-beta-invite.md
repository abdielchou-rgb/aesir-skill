# Æsir 故事织机 · 内测邀请

> 一个和你聊天的故事创作伙伴。
> 把模糊的想法变成完整的故事单元，像织布一样，一线一线织出属于你的故事。

---

## 设计理念

故事创作最难的不是"怎么写"，而是"不知道从哪开始"。大多数时候你只有一个模糊的想法——地铁上看到一个人、一个"如果"的假设、一种说不清的情绪。现有的写作工具要么太复杂（要学操作），要么太简单（生成完就没了）。

**Æsir 的设计理念是：故事是聊出来的，不是写出来的。**

你跟它说一个念头，它发散几条方向让你选，你选一条，它帮你收敛成一个完整的故事单元。每个单元有开头有结尾，但会留下几条未闭合的线索——你想继续的时候可以沿着线索往下走，不想继续的时候计入故事画布。

整个过程不需要学任何东西，用自然语言说话就行。

---

## 目标

**短期目标：** 让每个人都能在手机上把一个模糊想法变成一个完整的故事单元。不是片段，不是大纲——是能读的、有开头有结尾的故事。

**长期目标：** 让故事画布成为你的个人创作空间。画布里可以积累多个故事单元，它们之间可以有交叉、有伏笔、有世界共享。你可以随时回来继续任何一个单元，也可以把它们合并成更大的故事。

**非目标：** 不替代人创作。Æsir 做发散和收敛，你做选择和判断。它提供可能性，你决定方向。

### 基础用法

| 你说 | 它做什么 |

| "帮我想个故事" 或 输入一个模糊想法 | 发散 5 条世界线，让你选择 |
| "选第 3 条" | 收敛成 800-1200 字的完整故事单元 |
| "继续" / "往下走" | 沿未闭合线索延展下一段故事单元 |

## 后续可能性

### 近期（正在进行或计划中）

- **故事画布持久化：** 当前会话中的画布在关闭后消失。后续将结合 PWA 的 localStorage 实现跨会话保存

### 中期

- **多单元融合：** 故事画布中积累的多个单元可以合并，系统自动检测线索交叉点

### 长期

- **故事画布可视化：** 在 PWA 中以节点图形式展示画布中的所有单元和线索关系
- **协作创作：** 多个用户共享故事画布，各自延展不同线索
- **跨故事联想：** 系统在不同故事画布之间发现主题/人物/设定上的相似性，推荐融合

## 文件说明

收到两个文件：

1. **`aesir-coze-bot.json`** — Coze Bot 配置文件。在 Coze 平台（coze.cn）创建 Bot 后导入即可使用
2. **`aesir-coze-bot.md`** — 本文档的说明文件，包含所有工作流的完整内容

### 部署步骤（1 分钟）

1. 打开 coze.cn，注册/登录
2. 点击「创建 Bot」→ 填写 Bot 名称「Æsir 故事织机」
3. 点击右上角「导入配置」→ 选择 `aesir-coze-bot.json`
4. 在「模型选择」中选 Claude 3.5 Sonnet 或 GPT-4o
5. 点击「发布」→ 获得分享链接

安装后可以在手机 Coze app 中直接对话。
